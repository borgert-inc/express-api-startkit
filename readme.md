#### Nodemon
> npm install -g nodemon

#### Install packages
> npm install

#### Copy .ENV
> cp .env-example .env

#### Configure .env
> MONGO_DB=mongodb://localhost:27017/mongo_db

> APP_TOKEN=MyTokenForSession

#### Development (With nodemon watch .js and .twig)
> npm run dev

> http://localhost:3000

#### Eslint
> npm run eslint
